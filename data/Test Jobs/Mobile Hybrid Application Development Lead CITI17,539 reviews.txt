A hybrid mobile tech lead is an expert in building multi-channel / multi-platform Mobile and Tablet applications using Mobile application development frameworks such as React Native, Flutter, Kony etc.
The successful candidate is expected to be hands-on with Hybrid mobile application frameworks such as Cordova or React Native or Flutter, and will be responsible for success and timely delivery of software components and solutions.
The appropriate candidate will clearly demonstrate his ability to improve Continuous integration and provide process improvement areas.
The Role also includes
Design, development, testing and implementation of software solutions designed for mobile devices.
Scope for influencing design and architecture.
Liaise with multiple technology teams, operations and product contacts of varying degrees of seniority to facilitate project delivery.
Providing second line production support including root cause analysis for the developed software solutions.
UI Framework development
